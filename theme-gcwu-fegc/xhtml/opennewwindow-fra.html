Nouvelle fenêtre/Nouvel onglet - XHTML 1.0 Strict - Thème de la facilité d'emploi Web GC

The purpose of this jQuery plugin is to make all links which contain either the "external" or "internal" rel attribute to open in a new window and provide advance warning when JavaScript is enabled.

You should toggle Javascript on/off to view the differences. Please read the following for an explanation of each example.

External attribute

The following happens when rel="external" is used and JavaScript is enabled:

  • Opens the link in a new window
  • Adds the external link "pop-up" image after the link
  • Adds a hidden link description box to the link if one already exists (hidden link description box becomes visible when the link gains focus)
  • Adds "Lien externe" to the hidden link description box if it doesn't already exist
  • Adds "Ouvre dans une nouvelle fenêtre" to the hidden link description box

Examples

  1. No pre-existing "Lien externe" warning:

    Google (uses rel="external")

    Code: <a href="http://google.com" rel="external">Google</a>

  2. Pre-existing "Lien externe" warning:

    Google Lien externe (uses rel="external")

    Code: <a href="http://google.com" rel="external" class="cn-linkdesc">Google<span><span> Lien externe</span></span></a>

  3. Pre-existing hidden link description:

    Google Description du lien (uses rel="external")

    Code: <a href="http://google.com" rel="external" class="cn-linkdesc">Google<span><span> Description du lien</span></span></a>

Internal attribute

The following happens when class="wet-newwindow" is used and JavaScript is enabled:

  • Opens the link in a new window
  • Adds the internal link "pop-up" image after the link
  • Adds a hidden link description box to the link if one already exists (hidden link description box becomes visible when the link gains focus)
  • Adds "Ouvre dans une nouvelle fenêtre" to the hidden link description box

Examples

  1. No pre-existing hidden link description:

    Page "Pop-up" (class="wet-newwindow")

    Code: <a href="#" class="wet-newwindow">Page "Pop-up"</a>

  2. Pre-existing hidden link description:

    Page "Pop-up" Description du lien (class="wet-newwindow cn-linkdesc")

    Code: <a href="#" class="wet-newwindow cn-linkdesc">Page "Pop-up"<span><span> Description du lien</span></span></a>
